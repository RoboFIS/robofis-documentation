# Introducción

- **Nivel de acabado:** 7

RoboFIS es una aplicación de alquiler de robots. Tras registrarse, un usuario puede alquilar robots disponibles.

- **Tipos de robots:** limpieza, cocina y compras.
- **Funcionamiento:** los robots se desplazan a una dirección para completar tareas y, después, regresan a estaciones asignadas para cargar su batería.
- **Zona de operación:** los robots operan dentro de una zona asignada, definida como un polígono de coordenadas.

## Microservicios

- **Nuestro microservicio:** estado de robot
- **Otros microservicios:** notificaciones, usuarios, alquiler, stock
