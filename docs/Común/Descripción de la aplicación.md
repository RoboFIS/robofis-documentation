# Descripción de la aplicación

RoboFIS es una aplicación de alquiler de robots domésticos pensada para ofrecer una experiencia sencilla y accesible. Los usuarios pueden registrarse, elegir entre planes Básico, Intermedio o Premium, y utilizar un sistema de tokens para reservar robots de diversa tipología según su disponibilidad.

La plataforma cuenta con un dashboard intuitivo tanto para clientes como para empleados, desde el cual se gestionan usuarios, robots, notificaciones y zonas de operación. RoboFIS facilita el control del servicio, el seguimiento del estado de los robots y la comunicación con los usuarios, ofreciendo una solución completa y flexible para el alquiler de robots en el hogar.
